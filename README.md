# Regex → NFA → DFA Simulator

Interactive web-based simulator demonstrating concepts from
Theory of Computation.

Features
- Regular Expression → ε-NFA conversion (Thompson construction)
- ε-NFA → DFA conversion (subset construction)
- Graph visualization of automata
- Step-by-step string simulation
- State and transition highlighting
